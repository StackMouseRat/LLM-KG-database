# 第一节　高压电缆线路故障概述

随着城市建设的发展，电力电缆在城网供电中所占的比例也在增加，但随着电缆应用数量的增多及运行时间的延长，电缆故障也越来越多，故障的原因也复杂多样，主要可分为电缆本体故障和电缆附件故障两类。本节就35kV及以上电压等级电缆线路常见故障归纳如下。

## 1．电缆本体故障

（1）机械损伤。机械损伤故障是电缆本体故障中所占比例最大的一个，大约为60%，这类故障对电网影响大，造成的后果比较严重。  
1）直接外力作用造成的破坏，这方面的损坏主要由城市建设施工和交通运输造成。  
2）敷设过程对电缆的破坏，这类常见的是电缆因受力过大或者弯曲过度造成其局部绝缘能力降低或者电缆铠装层断裂。  
3）地面下沉。此现象往往发生在电缆穿越公路、铁路及高大建筑物时，由于地面的下沉而使电缆垂直受力变形，导致电缆破裂，甚至折断，或者造成电缆中间接头内部绝缘能力降低而发生击穿。  
4）自然力破坏。这个是不可抗拒的，主要由地质灾害或者温度太低冻坏电缆附件等造成，这个在电缆故障中所占比例较小。

（2）化学损伤。热化学作用，造成电缆过热的原因很多，如：  
1）过负荷是电缆过热的重要原因，在此情况下电缆绝缘内部气隙游离造成局部过热，从而使绝缘碳化，尤其是橡塑绝缘电缆在长期过热后，绝缘材料发生变硬、弹性消失进而出现裂纹等物理变硬。  
2）电缆沟或者电缆隧道内通风不良，电缆排布不合理，再加上长期电缆过负荷，很容易造成电缆过热的现象，使绝缘损坏加速。  
3）在热力管附近敷设电缆或热电同隧道架设，将大大缩短电缆的寿命。  
4）涡流影响。早期敷设电缆尤其是直埋电缆多穿蛇皮管，直立电缆多用穿钢管。当其中所穿电缆为三芯电缆时涡流现象不太明显（三相所成的相位差正好削弱了变化的磁场）；当所穿为单芯电缆时，在负荷高时有时能产生100℃左右的高温。  
5）绝缘受潮。它主要是在电缆后期运行中，电缆外护套有裂纹或者受外

## 2.3 外半导电层电场分布计算

为了分析该结构电缆的电场分布情况，对电缆内、外半导电层的电场进行了计算，计算过程如下：

按照均匀介质圆形导体单芯电缆电场强度进行计算，半径 \(x\) 处的电场强度计算公式为

$$
E_x=\frac{U}{x\ln\frac{r_i}{r_c}} \;(\mathrm{kV/mm})
$$

式中　\(U\)——导体对地电压，kV；

\(x\)——绝缘中任一点的半径，mm；

\(r_i\)——绝缘外半径，mm；

\(r_c\)——导体外半径，mm。

计算得：

最大电场强度　\(E_{\max}=\frac{U}{r_c\ln\frac{r_i}{r_c}}=5.74\ (\mathrm{kV/mm})\)

最小电场强度　\(E_{\min}=\frac{U}{r_i\ln\frac{r_i}{r_c}}=2.48\ (\mathrm{kV/mm})\)

由此可见最大电场强度出现在电缆内半导电层表面为5.74kV/mm，电缆外半导电层电场为2.48kV/mm。

## 2.4　电树枝产生原因分析

电树枝老化主要呈现两个阶段，即电树枝起始阶段和电树枝生长阶段。

电树枝起始阶段主要受产品缺陷影响，微孔、杂质、突起、气泡等缺陷是诱发电树枝的常见原因。3支故障电缆中间接头的电树枝均处于高压屏蔽电极的合模缝部位，在电树枝生长部位未发现气泡团，因此可排除气泡引发电树枝的可能性。由于合模缝部位需要人工进行砂纸打磨，打磨的光滑度、洁净度难以保证，因此分析合模缝部位打磨质量存在问题是导致在合模缝产生电树枝，进而引发在此部位发生击穿的主要原因。另外，人工打磨的合模缝质量分散性较大，不能排除同线路其他接头存在打磨质量、滋生电树枝的可能性。

国内众多学者对电树枝生长阶段的形态进行了研究，研究表明：“电树枝引发后在生长过程中表现出不同的形态特征，硅橡胶的电树枝形态主要分为3种，即树枝状电树、松枝状电树和丛状电树。树枝状电树仅有一支干或者少量树枝，树枝通道细小，生长速度缓慢。松枝状电树，生长速度较快，树枝具有明显的粗大松干通道，并旁支处多条分叉树枝，在分叉树枝上发展处密集可见的针状松叶。丛状电树枝形状为丛状，在电树枝起始后短时间内迅速形成，之后进入较长时间的滞涨期。电树枝形态与电压的频率有关，在工频电压下电树枝以树枝状和松枝状为主，在高频电压下以丛状电树枝为主，高频电压下树枝状和松枝状电树枝可向丛状电树枝转化”。

导电管，并在半导电管上缠绕多层绝缘带来增强绝缘，此种结构并未改善半导电套管端口处的电场分布，这可以很好地解释故障多发于此处的原因。金属应力锥与电缆绝缘之间既有 VDG 带，又有绝缘油来增强绝缘，在正常情况下可以确保终端安全稳定运行，但从现场解体来看，VDG 缠绕带有很多的裂口、翘起等隐患，这些缺陷的存在会在电缆与 VDG 带交界面产生气隙，气隙的放电电压通常较低，持续的局部放电进一步加剧缺陷的发展，最终导致电缆终端发生击穿故障。解体过程中，在电缆绝缘表面（非击穿孔附近）发现的黄色胶状物，即是局部放电留下的痕迹，黄色胶状物很好地印证了该应力锥与电缆绝缘之间存在多处局部放电，以及局部放电向击穿故障的演变过程。

2. 故障发生受环境温度影响原因分析

故障多发生于气温较低季节的发生可能原因推理如下：在温度较低时，局部放电产生的杂质、气体等不宜向周围扩散，形成局部累积，给局部放电提供了一定的有利条件，加剧局部放电的发展，最终形成贯穿性击穿故障。相反，温度较高时，生成的杂质、气体等缺陷易向周围扩散，一定程度上可以抑制局部放电程度，减缓其发展速度。

# 3 结论与建议

## 3.1　结论

本次 110kV 变压器电缆终端故障的发生，主要是由于其产品结构不能有效的均衡半导电管端口的电场分布，在 VDG 绝缘带材与电缆绝缘含有气隙的情况下，由局部放电逐渐发展成击穿故障。

## 3.2　建议

（1）建议加强对在运金属应力锥结构电缆终端采取局部放电检测手段，监测其局部放电情况，及时发现潜在的放电缺陷。

（2）鉴于该结构电缆终端在电场控制方面存在缺陷，已在多个网省地区发生多次故障，建议根据线路重要程度，分轻重缓急合理制订更换计划，逐步予以更换。

（1）A相户外终端外护套过热烧蚀现象为铝护套与脱焊的接地线金具之间产生长时间放电引起。发生放电的原因为该段电缆在接地线脱焊后其铝护套对接地线金具之间存在感应电压差。铝护套表面接地金具脱焊的原因为该户外终端长期存在进水受潮情况，铝护套表面锈蚀严重导致地线脱焊。

（2）电缆终端铜网端部与应力锥端部之间同样存在感应电压差，因此两者之间的电缆半导电层发生过热、烧蚀和放电现象；该类型户外终端在尾管底部位置主要靠环氧泥和热缩外护套进行防水，该类型设计可能存在防水能力不足的设计缺陷。

## 3　结论与建议

### 3.1　结论

综上所述，110kV户外终端温度异常现象是由电缆铝护套接地线脱焊引起，由于铝护套接地的消失，造成铝护套产生悬浮电位，进而产生感应电压压差并发生放电引起。

### 3.2　建议

电缆的接地系统在电缆线路安全稳定运行中起着重要作用，接地的破坏会导致严重的隐患，因此应及时开展接地电流检测、红外测温外观检查等，提前发现电缆接地异常，采取有效措施予以治理。
