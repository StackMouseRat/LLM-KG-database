# 对照试验题库

本目录用于存放“当前工作流 vs 裸 LLM / 消融版本”的对照试验材料。

当前已保存内容：

- `现有工作流生成链路梳理.md`
  - 基于前端实际入口、主流水线脚本和 FastGPT 已发布插件定义，整理当前生成链路。
- `现有工作流生成提示词.md`
  - 保存当前主流水线中与生成相关的实际提示词、模板切片逻辑和输入输出键。
- `并行生成_固定生成部分.md`
  - 单独固定当前“并行生成”插件的模型、system prompt 和输入契约，供对照试验复用。
- `对照试验题库_初稿.md`
  - 预留对照试验题目清单，已给出字段结构和首批候选题。
- `轻量实验_模板提示词.txt`
  - 轻量实验用的固定模板提示词，用于比较“裸 LLM”与“带模板 LLM”。
- `轻量实验_样题.txt`
  - 轻量实验首批样题，一行一题，可直接喂给批量对比脚本。

说明：

- 本目录只保存“可直接用于实验设计”的材料，不保存原始数据库 dump。
- 当前事实来源优先级：
  1. `frontend/` 中的实际前端调用链路
  2. `scripts/run_parallel_generation_pipeline.py`
  3. FastGPT Mongo 中 `app_versions` 的已发布插件配置
