## 2. fastGPT_json 节点体系核对（重点）
- `fastGPT_json/测试.json`：
- 7 个节点、5 条边，链路为：问题优化 -> 内容提取 -> 知识库检索 -> 清洗 -> 最终生成。
- `fastGPT_json/测试 Copy.json`：
- 14 个节点、12 条边，链路包含：缓存初始化（ifElse）-> 基础数据获取/清洗/更新 -> 计划器 -> 循环执行（loopStart -> 循环提取 -> HTTP -> 查询后处理 -> 变量更新 -> loopEnd）。
- 关键注意点：
- 两个 JSON 文件带 BOM，程序读取建议使用 `utf-8-sig`。
- 循环子节点依赖 `parentNodeId` 表示层级，不能只按顶层 edges 判断节点关系。
