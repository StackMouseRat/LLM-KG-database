# 前端开发文档

本目录用于存放论文演示版前端的开发说明。

当前前端已经具备以下页面与能力：

- 预案生成
- 图谱溯源
- 格式优化与质量评估
- 模板查看
- 登录与基础权限控制

当前仍未固化为成熟产品能力的部分：

- 历史记录管理
- 预案在线编辑
- 移动端适配
- 复杂工作流执行路径可视化

## 文档清单

- `frontend_current_status.md`：基于当前代码整理的前端现状、已实现功能、接口串接方式与能力边界。
- `frontend_todo.md`：按天拆解的开发任务清单
- `frontend_structure.md`：推荐目录结构与组件划分
- `frontend_api_types.md`：接口约定与数据类型定义

## 使用说明

- 如果要了解“当前代码实际已经做到什么”，优先看 `frontend_current_status.md`。
- `frontend_todo.md`、`frontend_structure.md`、`frontend_api_types.md` 主要反映前端设计与开发阶段的思路，其中部分内容已经落后于当前实现。
