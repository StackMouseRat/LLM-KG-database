## 四、案例文档内容特征

通过抽样检查，RAG 案例库文档具有以下特征：

### 1. 结构化案例格式
```
案例标题：具体设备型号 + 故障类型
1. 异常情况
   - 时间、地点、设备信息
   - 故障现象描述
   - 监控数据、测量结果
2. 分析处理
   - 原因分析
   - 处理步骤
   - 检查结果
3. 结论与预防措施
```

### 2. 包含实际数据
- 设备型号、参数
- 测量数据（电压、电流、温度等）
- 油色谱分析数据
- 图表、照片说明

### 3. 处理过程详细
- 逐步排查过程
- 使用的工具和方法
- 决策依据
- 最终解决方案

### 4. 经验总结
- 故障根本原因
- 预防措施
- 维护建议
- 注意事项
