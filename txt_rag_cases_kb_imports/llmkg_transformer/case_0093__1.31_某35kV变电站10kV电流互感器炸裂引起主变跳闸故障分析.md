# 1.31 某35kV变电站10kV电流互感器炸裂引起主变跳闸故障分析

### 一、故障情况

2013年11月25日21时01分43秒876毫秒，某35kV变电站1号主变比率差动出口；44秒556毫秒，该变电站2号主变低后备过电流Ⅱ段1时限及2时限、过电流Ⅲ段出口。

故障前该35kV变电站35kV母线为单母线运行，35kV某线路3511带全站负荷；1、2号主变并列运行，10kVⅠ母、Ⅱ母正常运行，母联100隔离开关在合，10kV某线路112运行于10kVⅠ母。

### 二、现场情况检查

（1）通过查询1号主变差动保护装置可以看到：

2013年11月25日21时01分43秒876毫秒，1号主变比率差动动作，差动电流为0.92A（差动定值为0.9A）。

（2）通过查询2号主变保护装置可以看到：

2013年11月25日21时01分44秒556毫秒，2号主变低后备过电流Ⅱ段1时限及2时限、过电流Ⅲ段动作故障电流为9.82A（过电流Ⅱ、Ⅲ段定值为6A，时间0.7s跳两侧断路器），故障电流现场测试数据见表1-50。

**表1-50 现场测试数据**

| 设备名称       | 2号主变         |
|----------------|------------------|
|                | 二次值           | 一次值     |
| 故障电流       | 9.82             | 392.8      |
| 102电流互感器变比 | 200/5            |            |

（3）通过检查1号主变本体、3501断路器及电流互感器、101断路器及电流互感器，发现101电流互感器A相发生炸裂。

（4）通过检查2号主变本体、3502断路器及电流互感器、102断路器及电流互感器，无异常。

（5）通过查看后台监控机信息，发现在此故障期间发生了10kV某线路112B相接地。

### 三、保护动作行为分析

（1）对该站1号主变保护动作行为分析如下：

通过现场查看，10kV某线路112发生B相接地故障，在检修人员前往处理该故障期间，1号主变101电流互感器A相因绝缘薄弱炸裂，导致A相接地，使故障发展为AB相间短路，1号主变差动保护动作跳开3501、101断路器，动作行为正确。

（2）对该站2号主变保护动作行为分析如下：

由于1号主变101电流互感器位于10kV母线侧，当1号主变差动保护动作跳开101、3501断路器后故障并未隔离，2号主变低后备保护过电流Ⅱ、Ⅲ段动作跳开3505、102断路器，动作

### 四、整改措施及建议

（1）对该站 1、2 号主变低压侧电流互感器进行更换，并联系生产厂家分析电流互感器炸裂原因。

（2）加强变电站设备的运维与管理。
