# 电力电缆故障层级梳理（第一层级）

## 1. 目标

本文用于重新确定电力电缆故障图谱的第一层级，也就是根节点下的一级故障大类。

本版按“不重不漏”原则收敛分类：二级层只保留故障结果或故障对象，受潮、老化、过电压、制造、安装、设计、诊断定位等原因或过程信息不再与故障并列。

## 2. 根节点

- 根节点名称：`电力电缆`

## 3. 第一层级推荐结果

当前建议将电力电缆故障第一层级定为以下 `5` 个一级大类：

1. 绝缘系统故障
2. 机械损伤故障
3. 接头与附件故障
4. 热故障
5. 火灾事故

## 4. 推荐层级结构

```text
电力电缆故障
├─ 绝缘系统故障
├─ 机械损伤故障
├─ 接头与附件故障
├─ 热故障
└─ 火灾事故
```

## 5. 合并说明

- `绝缘劣化与击穿故障` 合并来源：`纸绝缘电缆绝缘损坏故障`、`绝缘受潮故障`、`绝缘老化故障`、`过电压击穿故障`、`交联聚乙烯电缆进水故障`
- `机械外力损伤故障` 合并来源：`机械损伤故障`、`敷设损伤故障`
- `接头与终端附件故障` 合并来源：`中间接头绝缘击穿故障`、`电缆头制作质量故障`、`产品质量缺陷故障`、`设计不良故障`
- `过负荷过热故障` 合并来源：`过热故障`、`过负荷运行故障`
- `电缆火灾故障` 合并来源：`电缆火灾故障`
- 原 `故障性质判断异常`、`故障定位异常` 不再作为二级故障节点，已并入各故障的应对措施、应急资源和诊断定位要求。

## 6. 一级大类说明

### 1. 绝缘系统故障

- 对应二级故障：`绝缘劣化与击穿故障`
- 合并来源：`纸绝缘电缆绝缘损坏故障`、`绝缘受潮故障`、`绝缘老化故障`、`过电压击穿故障`、`交联聚乙烯电缆进水故障`

### 2. 机械损伤故障

- 对应二级故障：`机械外力损伤故障`
- 合并来源：`机械损伤故障`、`敷设损伤故障`

### 3. 接头与附件故障

- 对应二级故障：`接头与终端附件故障`
- 合并来源：`中间接头绝缘击穿故障`、`电缆头制作质量故障`、`产品质量缺陷故障`、`设计不良故障`

### 4. 热故障

- 对应二级故障：`过负荷过热故障`
- 合并来源：`过热故障`、`过负荷运行故障`

### 5. 火灾事故

- 对应二级故障：`电缆火灾故障`
- 合并来源：`电缆火灾故障`
