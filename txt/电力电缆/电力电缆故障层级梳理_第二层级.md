# 电力电缆故障层级梳理（第二层级）

## 1. 目标

本文在第一层级基础上，按“不重不漏”原则整理电力电缆故障图谱第二层级，即：

`电力电缆 -> 一级故障大类 -> 二级故障小类`

## 2. 第二层级整理原则

- 二级节点只保留故障结果或故障对象，不把原因、诱因、检测过程、处置过程作为同级故障
- 受潮、进水、老化、过电压、制造缺陷、安装缺陷、设计不良等统一进入故障原因层
- 电缆本体、接头、终端和附件的制造、安装、设计质量信息统一汇入相关故障描述，不再拆成多个二级项
- 原诊断定位类信息不作为设备故障二级项，保留到应对措施和应急资源中

## 3. 第二层级推荐结果

当前建议电力电缆第二层级收敛为以下 `5` 个二级小类。

### 3.1 绝缘系统故障

1. 绝缘劣化与击穿故障

### 3.2 机械损伤故障

1. 机械外力损伤故障

### 3.3 接头与附件故障

1. 接头与终端附件故障

### 3.4 热故障

1. 过负荷过热故障

### 3.5 火灾事故

1. 电缆火灾故障

## 4. 推荐层级结构

```text
电力电缆故障
├─ 绝缘系统故障
│  └─ 绝缘劣化与击穿故障
├─ 机械损伤故障
│  └─ 机械外力损伤故障
├─ 接头与附件故障
│  └─ 接头与终端附件故障
├─ 热故障
│  └─ 过负荷过热故障
└─ 火灾事故
   └─ 电缆火灾故障
```

## 5. 合并映射

- `绝缘劣化与击穿故障` 合并来源：`纸绝缘电缆绝缘损坏故障`、`绝缘受潮故障`、`绝缘老化故障`、`过电压击穿故障`、`交联聚乙烯电缆进水故障`
- `机械外力损伤故障` 合并来源：`机械损伤故障`、`敷设损伤故障`
- `接头与终端附件故障` 合并来源：`中间接头绝缘击穿故障`、`电缆头制作质量故障`、`产品质量缺陷故障`、`设计不良故障`
- `过负荷过热故障` 合并来源：`过热故障`、`过负荷运行故障`
- `电缆火灾故障` 合并来源：`电缆火灾故障`
- 原 `故障性质判断异常`、`故障定位异常` 不再作为二级故障节点，已并入各故障的应对措施、应急资源和诊断定位要求。
