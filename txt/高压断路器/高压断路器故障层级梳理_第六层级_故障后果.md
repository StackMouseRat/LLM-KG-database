# 高压断路器故障层级梳理（第六层级：故障后果）

## 1. 目标

本文在现有第一至第五层级的基础上，统一为全部二级小类补充第六层级，即：

`高压断路器 -> 一级故障大类 -> 二级故障小类 -> 故障后果`

当前采用单节点设计：

- 每个二级故障小类下，只保留 1 个 `故障后果` 节点
- 原文未系统单列本层内容
- 本稿根据前面各层内容及设备运行影响做最小归纳，尽量不超出原文逻辑

## 2. 命名规则

- 文档层级名称统一写作：`故障后果`
- 图谱落地时的唯一节点名建议写作：`<二级故障小类>-故障后果`

## 3. 第六层级总结构

```text
高压断路器故障
└─ 一级故障大类
   └─ 二级故障小类
      └─ 故障后果
```

## 4. 各二级小类的故障后果

### 4.1 灭弧与介质状态故障

#### 油断路器缺油故障

- 第六层级节点：`故障后果`
- 唯一节点名建议：`油断路器缺油故障-故障后果`
- 描述/说明：
  主要后果是灭弧能力和开断安全性显著下降，严重时断路器不能安全切断电路，可能进一步引发设备损坏、停运和事故扩大。

#### 空气断路器气压降低故障

- 第六层级节点：`故障后果`
- 唯一节点名建议：`空气断路器气压降低故障-故障后果`
- 描述/说明：
  主要后果是操作回路被切断或重合闸被闭锁，断路器运行能力下降，无法可靠执行分合闸操作，影响供电可靠性和故障处置能力。

#### 六氟化硫介质泄漏故障

- 第六层级节点：`故障后果`
- 唯一节点名建议：`六氟化硫介质泄漏故障-故障后果`
- 描述/说明：
  主要后果是绝缘和灭弧性能下降，断路器被迫停运或限运；大量漏气时还会带来人员暴露风险和现场处置难度增加。

#### 灭弧室烧损爆炸故障

- 第六层级节点：`故障后果`
- 唯一节点名建议：`灭弧室烧损爆炸故障-故障后果`
- 描述/说明：
  主要后果是灭弧室严重损坏甚至爆炸，导致设备报废、停电范围扩大，并伴随人身伤害和次生事故风险。

#### 触头及导电连接异常故障

- 第六层级节点：`故障后果`
