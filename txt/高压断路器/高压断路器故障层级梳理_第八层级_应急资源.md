# 高压断路器故障层级梳理（第八层级：应急资源）

## 1. 目标

本文在现有第一至第七层级的基础上，统一为全部二级小类补充第八层级，即：

`高压断路器 -> 一级故障大类 -> 二级故障小类 -> 应急资源`

当前采用单节点设计：

- 每个二级故障小类下，只保留 1 个 `应急资源` 节点
- 原文未系统单列本层内容
- 本稿根据各类故障的典型处置流程和检修要求做最小归纳

## 2. 命名规则

- 文档层级名称统一写作：`应急资源`
- 图谱落地时的唯一节点名建议写作：`<二级故障小类>-应急资源`

## 3. 第八层级总结构

```text
高压断路器故障
└─ 一级故障大类
   └─ 二级故障小类
      └─ 应急资源
```

## 4. 各二级小类的应急资源

### 4.1 灭弧与介质状态故障

- 第八层级节点：`应急资源`
