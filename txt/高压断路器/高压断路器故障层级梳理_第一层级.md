# 高压断路器故障层级梳理（第一层级）

## 1. 目标

本文用于先确定高压断路器故障图谱的第一层级，也就是根节点下的一级故障大类。

当前分类以 `txt/高压断路器/高压断路器.txt` 为依据，先做适合后续图谱展开的一级分类，不直接照搬原文章节标题。

## 2. 根节点

- 根节点名称：`高压断路器`

## 3. 第一层级推荐结果

当前建议将高压断路器故障第一层级定为以下 `7` 个一级大类：

1. 灭弧与介质状态故障
2. 导电回路与开断故障
3. 合分闸操作故障
4. 操动机构与机械传动故障
5. 操作能源与控制回路故障
6. 继电保护与测量二次回路故障
7. 本体绝缘与支撑结构故障

## 4. 推荐层级结构

```text
高压断路器故障
├─ 灭弧与介质状态故障
├─ 导电回路与开断故障
├─ 合分闸操作故障
├─ 操动机构与机械传动故障
├─ 操作能源与控制回路故障
├─ 继电保护与测量二次回路故障
└─ 本体绝缘与支撑结构故障
```

## 5. 互斥边界说明

为保证后续第二层级尽量不重不漏，当前先固定以下边界：

- `合分闸操作故障` 只收“误跳、误合、拒合、拒分”这类直接功能表现，不收它们的原因项
- `操作能源与控制回路故障` 只收直流电源、控制回路、线圈、辅助开关、液压/油风管路等执行链故障
- `继电保护与测量二次回路故障` 只收保护整定、互感器回路、保护出口、寄生回路等保护域问题，不再与控制回路混放
- `操动机构与机械传动故障` 只收操动机构内部的铁心、挂扣、滚子、掣子、轴销、连板、拉杆、储能装置等传动部件故障
- `本体绝缘与支撑结构故障` 只收瓷件、基础、支架、本体连接件、绝缘拉杆等非操动机构主传动件故障

## 6. 一级大类说明

### 1. 灭弧与介质状态故障

建议纳入以下内容：

- 油断路器缺油
- 空气断路器气压降低
- 六氟化硫断路器漏气、压力报警、气体外逸
- 灭弧室烧损、爆炸风险
- 与灭弧介质状态直接相关的运行异常

说明：

- 这一类重点覆盖油、气、SF6 等灭弧或绝缘介质的状态异常
- 真空灭弧室相关异常后续也建议优先归到本类或“导电回路与开断故障”

### 2. 导电回路与开断故障

建议纳入以下内容：

- 断路事故
- 开断能力异常
- 导电回路直阻超标
- 触头、导电夹、拐臂接触异常
- 影响导通和开断能力的本体故障

说明：

- 这一类强调“电流是否能可靠导通和切断”
- 后续如果需要细分，可拆出“导电异常”和“开断异常”两个二级方向

### 3. 合分闸操作故障

建议纳入以下内容：

- 误动
- 拒动
- 合分闸动作异常
- 分、合闸后动作不符合预期的功能性故障

说明：

- 这是后续最核心的一类，建议优先展开二级小类
- 原文 `2.2`、`2.3`、`2.4` 的主要内容都应汇入本类

### 4. 操动机构与机械传动故障

建议纳入以下内容：

- 机构卡涩
- 合分闸铁心动作失灵
- 储能机构不能储能
- 滚子、掣子、杠杆、连板、轴销等传动部件异常
- 偏卡、拉杆折断、卡阻、死点等机械问题

说明：

- 这一类强调“机构能不能把电气命令转成可靠机械动作”
- 真空断路器储能机构相关问题也优先归到本类
- 只覆盖操动机构内部传动链，不覆盖本体瓷件、基础、支架等非传动结构

### 5. 操作能源与控制回路故障

建议纳入以下内容：

- 直流电源电压过低或过高
- 液压机构失压
- 操作油风管路异常
- 控制回路短路
- 控制回路接地
- 合闸回路故障
- 分闸回路故障
- 合闸接触器、分闸线圈、合闸线圈异常
- 辅助开关、转换接点异常

说明：

- 原文中大量关于电压、线圈、接触器、控制回路、液压和油风管路的内容都应纳入本类
- 这一类与“合分闸操作故障”关系紧密，但前者是原因域，后者是结果域

### 6. 继电保护与测量二次回路故障

建议纳入以下内容：

- 继电保护误动作
- 保护整定不当
- 互感器回路故障
- 寄生回路
- 出口继电器线圈接地
- 与二次系统相关的误跳、误合诱因

说明：

- 这一类建议单独保留，不并入控制回路故障
- 原因是继电保护、互感器回路和保护出口回路在运维上通常需要单独管理

### 7. 本体绝缘与支撑结构故障

建议纳入以下内容：

- 绝缘不良事故
- 瓷件、支架、连接件损伤
- 部件弯曲、变形、折断
- 基础支架不牢、连接松动
- 绝缘拉杆拉脱
- 其他本体结构强度不足或绝缘性能下降问题

说明：

- 这一类用于承接原文中的绝缘问题和本体支撑结构损伤问题
- 为避免与“操动机构与机械传动故障”重叠，本类不再收操动机构内部传动件

## 8. 当前不建议直接升为第一层级的项

以下内容当前不建议直接作为一级大类：

- `真空断路器常见故障`
- `油断路器故障`
- `空气断路器故障`
- `六氟化硫断路器故障`
- `预防技术措施`

原因：

- 前四项更像设备类型或灭弧介质类型，不是统一口径的故障大类
- “预防技术措施”属于措施层，不宜和故障层并列

## 9. 下一步建议

第一层级确认后，建议按以下顺序继续：

1. 先为 `合分闸操作故障` 细化第二层级
2. 再展开 `操作能源与控制回路故障`
3. 然后补 `灭弧与介质状态故障`、`操动机构与机械传动故障`
4. 最后整理各一级大类对应的原因、现象、措施、后果等详情层
